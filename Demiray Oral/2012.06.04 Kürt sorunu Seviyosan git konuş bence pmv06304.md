# Kürt sorunu: Seviyosan git konuş bence

*Demiray Oral 04.06.2012*

<div class="yazi"><p>Bir arkadaş tavsiyesidir, daha doğrusu arkadaş gazlamasıdır, <i>“Seviyosan git konuş bence”</i> lafı.</p>
<p>Anonimdir, söylenene söylene klişenin şahikası mertebesine yükselmiştir.</p>
<p>Kıza vurulmuş ama yanına bir türlü yaklaşamayan, reddedilme korkusunu kişilik problemi yapan çaresiz erkek, yakın arkadaşından derdine derman dilenir.</p>
<p>Yakın arkadaş önce sorar, <i>“Gerçekten seviyo musun oğlum sen bu kızı?”<br/><br/></i><i>“Evet”</i> cevabını alınca adeta hayatın manasını açıklıyormuş misali yapması gerekeni söyler: <i>“Abi seviyosan git konuş bence...”</i> </p>
<p>Karşısındaki <i>“Konuşayım di mi”</i> şeklinde tereddüt içinde ikiletince de finali yapar: <i>“Ne kaybedersin ki...”</i></p>
<p>Başbakan Erdoğan hafta sonunda gittiği Diyarbakır’da, <i>“2005’te Diyarbakır’da ne dediysek aynen, bire bir arkasındayız”</i> demiş.</p>
<p>Bunu gazeteden okuyunca aklıma geliverdi işte.</p>
<p>O zamanlar Erdoğan tıpkı arkadaşına açılan dertli âşık misali topluma, yani bize sormuştu.</p>
<p>Kürt sorununu çözmek istiyorum ama ne yapacağımı bilemiyorum, diye.</p>
<p>Biz ona <i>“Gerçekten seviyo mu devlet artık Kürtleri”</i> diye sormaya bile gerek duymamıştık.</p>
<p>Çünkü o zamanki Erdoğan devleti dönüştürmeyi, demokratikleştirmeyi isteyen bir siyasetçiydi.</p>
<p>Türkler için de, Kürtler için de umuttu.</p>
<p>Onun dönüştürmeye çalıştığı yeni devletin Kürtleri gerçekten seveceğini, o devlette Kürtlerin Türklerle eşit olacağını hayal ediyorduk.</p>
<p>Bu yönde çok alametler belirmişti.</p>
