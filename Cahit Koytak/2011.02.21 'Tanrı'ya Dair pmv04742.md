# ‘Tanrı’ya Dair 

*Cahit Koytak 21.02.2011*

<div class="yazi"><p>Gözyaşının, nasıl, akmak için göze,<i></i> <br/>Gülmenin, ağlamanın, nasıl <br/>                         görünür olmak için yüze, <br/>Susuzluğun suya, hasretin uzaklığa, <br/>Sevginin doğmak için, nasıl <br/>                                 gönle ihtiyacı varsa, <br/>Varlığın da, <i>olmak</i> için, öyle, <br/>                          Tanrı’ya ihtiyacı vardır. <br/><br/>Kokunun, saçılmak için, nasıl <br/>                                         çimene, çiçeğe, <br/>Rengin ışığa, ışığın güneşe, <br/>Güneşin, kendini faş etmek için, nasıl <br/>        sırlarla dolu geceye ihtiyacı varsa, <br/>Varlığın da, <i>olmak</i> için, öyle, <br/>                          Tanrı’ya ihtiyacı vardır. <br/><br/>Ezginin, nasıl, duyulmak için sese <br/>Ve sesin çalgıya, çalgının nefese, <br/>Şiirin, gönülden kopmak için, nasıl, <br/>                 bilincin altına, bilincin üstüne, <br/>Aklın düzüne, aklın tersine, <br/>Hayatın hem gündeliğine, <br/>Hem geriliğine ihtiyacı varsa, <br/>Varlığın da, <i>var</i> <i>olmak</i> için, öyle, <br/>                           Tanrı’ya ihtiyacı vardır. <br/><br/>İlhamın, vahiy olmak için, nasıl <br/>           daha yüksek, daha kusursuz, <br/>                         ve daha sönümsüz söze; <br/>Fikrin, iman olmak için, nasıl <br/>            daha büyük, daha ışıltılı, <br/>                     daha issiz, dumansız köze; <br/>Ve hayatın, sanat olmak için, nasıl, <br/>                        ne uyku, ne ölüm tanıyan <br/>Daha büyük, daha karmaşık, <br/>                   daha baş döndürücü düşe <br/>                                               ihtiyacı varsa, <br/>İnsanın da, <i>varolmak</i> için, öyle, <br/>                            Tanrı’ya ihtiyacı vardır. <br/><br/>10 Aralık 2007 <b><i><br/>Yeni Başlayanlar İçin Metafizik</i> Kitabı<br/><br/></b>***<b><i></i></b> <b><i> </i></b> <br/></p>
<h4>‘Kitap’a Dair </h4>
<p>Hangi kitabı okurken, ruhun, <br/>Çeşmenin önündeki kap gibi <br/>Tanrıyla dolup taşıyor – ya da <br/>Yağmurun altında kül gibi <br/>Sevgiden ve erinçten eriyip akıyorsa, <br/>Korkma, oku o kitabı, <br/>Korkma, o kitap sana indirildi.   <br/><br/>27 Kasım 2009 <b><i><br/>Şen Maneviyat</i> Kitabı</b></p>
